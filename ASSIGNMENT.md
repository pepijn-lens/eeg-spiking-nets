# Assignment 2: Create a precise control-dataset to evaluate a single hypothesis of a ML/DL paper

**Weight:** 15% of the grade

## Task

Students will submit a blog post with:
- A motivation of why their control dataset precisely tests a particular property,
  where the property is motivated with a link to one or multiple relevant papers.
- A description of the control dataset with some examples.
- A description of how the dataset was generated.

The submission must include links to the dataset and the code used to generate the
dataset (e.g. on GitHub/GitLab).

---

## Rubric: Penalties for Writing (applies to this blog)

- Using a term before defining/motivating it.
- Too much unnecessary detail: words can be removed without significantly changing
  the text. Each word should have a reason to be there.
- Unclear logical reasoning step.
- Inconsistent use of terminology. Use a single term for a single concept; 1-to-1.
- Writing too verbose / full sentences where bullets suffice.
- The text is not stand-alone; it is not peer-understandable.
- Not following the writing guidelines: https://jvgemert.github.io/writing.pdf

### Key writing guidelines to self-check against
- W1: Terms/concepts must be motivated before they are used.
- W2: Each paragraph has a single topic and a conclusion answering "So what?".
- W3: Each sentence continues on the topic where the previous one ends.
- W4: Figure captions explain how to read the figure; all symbols/terms are explained.
- W5: Figures/tables state their conclusion in the caption — what should the reader see?
- Scale scores between 0 and 100, not 0 and 1.
- Avoid "very", "performance" (ambiguous), reference words across paragraphs.
- Figures: label all axes with units, use legends and titles, end caption with a conclusion.

---

## Scope reminder (for this project)

This assignment is the **control dataset only**, testing **one** hypothesis.
Out of scope here: energy recomputation, real-EEG reproduction, and ablation results.
The dataset must *enable* those later, but they are not deliverables for Assignment 2.
